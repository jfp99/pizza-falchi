## Instructions de Téléchargement

### Étape 1: Visiter les sources
1. Ouvrir les URLs Unsplash/Pexels ci-dessus
2. Parcourir les résultats pour trouver l'image la plus appétissante
3. Cliquer sur l'image pour voir en haute résolution

### Étape 2: Télécharger
- **Unsplash**: Cliquer sur "Download free" (bouton vert)
- **Pexels**: Cliquer sur "Free Download" puis choisir taille "Large"
- **Images produits**: Clic droit > "Enregistrer l'image sous..."

### Étape 3: Optimiser
Si l'image dépasse 500KB:
- Utiliser https://tinypng.com/ pour compresser
- Ou utiliser le script de compression fourni

### Étape 4: Renommer et placer
- Renommer selon le nom indiqué (ex: `fromage.jpg`)
- Placer dans le dossier destination exact
